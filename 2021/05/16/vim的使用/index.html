vim的使用

  • 各种插入模式

  • 简单的移动光标

  • 拷贝/粘贴 （注：p/P都可以，p是表示在当前位置之后，P表示在当前位置之前）

  • undo/redo

  • 打开/保存/退出/改变文件(Buffer)

  • . → (小数点) 可以重复上一次的命令

  • 让光标移动更有效率

    NG → 到第 N 行 （注：: N 到第N行，如 :137 到第137行）
    gg → 到第一行。（注：相当于1G，或 :1）
    G → 到最后一行。
    按单词移动：

  • % : 匹配括号移动，包括 (, {, [. （你需要把光标先移到括号上）

  • 和 #: 匹配光标当前所在的单词，移动光标到下一个（或上一个）匹配单词（*是下一个，#是上一个）
  •  w → 到下一个单词的开头。
e → 到下一个单词的结尾。
